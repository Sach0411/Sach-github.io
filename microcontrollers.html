Linux · Self-Hosted

Home Server

Repurposed an old desktop PC into an always-on home server running Linux. Deployed AdGuard Home as a network-wide DNS blocker to eliminate ads and trackers on every device on the network — no client-side configuration needed. Also runs Plex as a personal media server, streaming movies and TV to any device on the local network. The whole setup runs headless 24/7, managed remotely over SSH.
